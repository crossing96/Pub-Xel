# Pub-Xel Worksheet

Your **Pub-Xel Worksheet** is a normal Excel file (`.xlsx`) for listing articles.

---

## Create one

**File → New Excel Template** → choose where to save.

The first time, you get a simple starter file. Later, new worksheets use the columns you picked in **Preferences → Worksheet**.

---

## Columns

Turn columns on or off in **Preferences → Worksheet**. **Ref** (PMID) is always included.

Common columns: **Title**, **Journal**, **AuthorYear**, **Abstract**, **Citation**, **IF2025** (impact factor).

Each row is one article. Put the PMID in **Ref**, then use **Import PubMed Data** to fill the rest.

---

## Open a recent worksheet

**File** menu lists worksheets you opened before — click one to jump to its folder in File Explorer.

---

## Drop a file on Pub-Xel

- **`.xlsx`** — adds it to recent worksheets
- **`.nbib`** — import from PubMed export ([details](Importing-nbib-Files.md))

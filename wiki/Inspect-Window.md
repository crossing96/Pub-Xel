# Inspect Window

**Role:** See what you copied — which PDFs exist, PubMed details, and quick actions.

Open it: copy IDs → **Inspect Files** or **Alt+Shift+J**.

---

## What you see

- List of IDs and how many have PDFs
- **Main files** — your article PDFs, **PMID.pdf** (e.g., `33301246.pdf`)
- **Supplementary files** — extra files for the same article ([naming guide](Supplementary-Files.md))
- PubMed info (title, etc.) when online

---

## Buttons

| Button | What it does |
|--------|----------------|
| **All** | Open every PDF listed |
| **Selected** | Open checked PDFs only |
| **Export All / Selected** | Copy PDFs to your Output folder |
| **View PubMed** | Open PubMed in browser |
| **Make Worksheet** | Save a new Excel file filled with PubMed info |
| **Exit** | Close |

Press **Esc** to close.

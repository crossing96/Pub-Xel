# Pub-Xel Wiki

**Pub-Xel** helps you manage biomedical articles with **Microsoft Excel** and **PubMed** — fill in citation info, find your PDFs, and open them quickly.

Website: [pubxel.org](https://pubxel.org/)

---

## What Pub-Xel does


| Features               |                                                                                 |
| ---------------------- | ------------------------------------------------------------------------------- |
| **Excel worksheet**    | Your articles listed in Excel (titles, journals, citations, etc.)               |
| **Import PubMed Data** | Fill Excel cells with PubMed data                                               |
| **Open Files**         | Open PDFs directly from Excel or other sources                                  |
| **Inspect Files**      | Review articles, open PDFs, export files, and more, from Excel or other sources |
| **Hotkeys**            | Access functionalities using global hotkeys.                                    |


---

## Requirements

- **Windows** (macOS version not available yet)
- **Microsoft Excel**

## Quick start

1. [Install Pub-Xel](Installation.md) (Windows).
2. Set **Main library** and **Output** in **Preferences**.
3. **File → New Excel Template** — save your first worksheet.
4. In the worksheet, enter PMIDs and click **Import PubMed Data** to directly import PubMed data to Excel.
5. Download PDFs from PubMed and save them in your **Main library** as **PMID.pdf** (e.g., `33301246.pdf`).
6. Copy PMIDs in Excel and press **Alt+Shift+K** to open matching PDFs.

More detail: [Getting Started](Getting-Started.md)

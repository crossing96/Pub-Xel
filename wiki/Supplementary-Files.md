# Supplementary Files

**Role:** Extra files for an article — supplements, figures, slides, etc. — separate from the main PDF.

Save them in the same **Main library** (or secondary library) as your article PDFs.

---

## Main file vs supplementary file


| Type                   | Naming                    | Example                                                                        |
| ---------------------- | ------------------------- | ------------------------------------------------------------------------------ |
| **Main file**          | **PMID.pdf**              | `33301246.pdf`                                                                 |
| **Supplementary file** | **PMID** + `_` + **text** | `33301246_supplement.pdf` `33301246_supplfigure1.jpeg` `33301246_tableS1.xlsx` |


---

## How to name supplementary files

Use the PMID, then an underscore `_`, then a short label (letters, numbers, or both).

**Examples:**

- `33301246_supplement.pdf`
- `33301246_figure2.pdf`
- `33301246_slides.pptx`
- `33301246_protocol.docx`

Pub-Xel links these to the same PMID as the main article.

---

## Inspect window

When you open [Inspect Window](Inspect-Window.md), supplementary files appear under **Supplementary files**, separate from **Main files**.

You can open or export them individually, or together with the main PDF.

---

## Tip

Keep the main article as **PMID.pdf** (e.g., `33301246.pdf`). Use `_` only for extra files so Pub-Xel can tell them apart.

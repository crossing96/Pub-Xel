# Global Hotkeys

Use these shortcuts while Pub-Xel is running — even if the window is hidden in the system tray.

Change them in **Preferences → Hotkeys** (restart after saving).

---

## Defaults


| Action        | Keys            |
| ------------- | --------------- |
| Open Files    | **Alt+Shift+K** |
| Inspect Files | **Alt+Shift+J** |
| View PubMed   | **Alt+Shift+P** |


---

## Typical use

1. Copy PMIDs in Excel (**Ctrl+C**).
2. Press **Alt+Shift+K** — your PDFs open.

---

## Change hotkeys

**Preferences → Hotkeys** → pick a modifier and one letter for each action → **Save and Exit** → restart Pub-Xel.

Each action needs a **different letter**. **Restore Default** puts back K, J, and P.

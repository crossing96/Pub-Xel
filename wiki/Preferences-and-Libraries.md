# Preferences and Libraries

**File → Preferences** → change settings → **Save and Exit** → restart when asked.

---

## Library


| Setting                | What it’s for                                          |
| ---------------------- | ------------------------------------------------------ |
| **Main library**       | Folder where you keep PDFs — **PMID.pdf** (e.g., `33301246.pdf`) |
| **Output**             | Where exported files go (must be a different folder)   |
| **Secondary library**  | Extra folders to search if a PDF isn’t in Main library |
| **Include subfolders** | Also look inside subfolders of Main library            |
